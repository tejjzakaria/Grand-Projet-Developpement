■ファイル名 now_loading08
■ファイルサイズ 4.0K
■初期サイズ 200×80
■イメージ
クリーンなイメージのロードムービーです。
■使用方法
設置したいムービーの前にLoading_MCムービークリップを設置します。1フレーム目にLoading_MCを2フレーム目に読み込ませたいムービーを配置します。ムービーをスタートすると2フレーム以降のムービーを読み込み終わるまで、1フレーム目に配置されたLoading_MCが表示されます。
■プレビューについて
このムービーはインターネット回線越しにデータを読み込むまでのロードムービーなので、読み込む対象がすでにあるローカル上（自分のパソコン上）では上手く動作しません。サーバにアップロードして回線越しにご確認下さい。また、お使いの回線が光ファイバーやADSLなどの高速回線の場合は、読み込む容量によっては一瞬しか表示されない場合もございますので、あからじめご了承下さい。
■バックグラウンド色の変更
ムービープロパティを開いて背景色を変更してください。
