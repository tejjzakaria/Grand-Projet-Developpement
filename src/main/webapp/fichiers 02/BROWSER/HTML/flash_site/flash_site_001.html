■サイトの構成と編集

＜サイトの構成＞
シーン１に配置されている、"ALL"というムービークリップは、ライブラリ内、「メイン」というフォルダの中にあります。
この "ALL" の中には、Opening ムービーをはじめ、各ページのコンテンツ（ムービークリップ）のすべてが含まれています。
各コンテンツは、ライブラリ内にそれぞれフォルダに分けて収納されていますので、編集の際にはそれぞれの該当ファイルを開き編集を行って下さい。

＜テキストの変更＞
サイトのタイトル変更は、ライブラリ内、「メイン」というフォルダ内にある、"Site title" というシンボルを編集する事で変更可能です。 このサンプルサイトでは最も一般的と思われる、Top、About、Gallery、Diary、BBS の５ページを用意していますが、ページのタイトルを変更したい場合は、各ページごとのフォルダ内、「○○用テキスト」というフォルダの中にある、"Sub title 00" というシンボルを編集する事で変更可能です。

また、それに伴って、各メニューボタンの名称を変更する必要が生じた場合は、ライブラリ内、「メインメニューボタン」フォルダの "main button 用1〜6" をそれぞれ編集する事で変更可能です。

その他のテキストも同様にそれぞれライブラリ内の該当するシンボルを編集する事で、変更可能です。

＜各ボタンへのリンクの付け方＞
各ムービー上、またはグラフィックシンボル上に配置されているボタンシンボルを選択し、それぞれにアクションを指定して下さい。

＜背景写真の変更＞
各ページ、オープニングで、使われているイメージ（写真）は任意のものへ変更可能です。
変更の方法ですが、ここでは "About"を例にご説明いたします。

ライブラリ内、Aboutフォルダを開くと、さらに「About用画像」というフォルダがあります。このフォルダ内にある、「About用背景」を開き、任意の画像を張り付ける事で、画像の変更出来ます。
画像のサイズは、420×277 ピクセルとなっています。違うサイズの画像を張り付けると、各ページはもちろん、オープニングでもレイアウトが崩れますので、ご注意ください。また、画像を張り付ける位置にも注意が必要です。
元の画像と全く同じ位置に張り付けないと、これもレイアウトが崩れる結果となってしまいますので、フラッシュのガイドを利用するか、元の画像が張られている、座標を参照するなどして張り付けて下さい。

＜Galleryについて＞
Galleryでは1ページで、36コの画像を参照できる構成となっています。
Gallery内、左に配置された四角のボタンにマウスを重ねる事で、ページの右側に画像が表示されます。表示される画像のサイズは
220×165となっていますが、もっと大きなサイズで画像を見せたい場合は、各ボタンにリンクをつける事で通常のHTMLで見せるという方法もあります。
画像はそれぞれボタンシンボルの中に含まれており、ライブラリ → Gallery用 → Gallery用ボタン 内の gallery_button_001〜036 にそれぞれ配置されています。（現在、参照している画像は、同じ階層の Gallery用画像 というフォルダ内におさめられています。）画像を変更するには各ボタンシンボルを開き、任意の画像に張り替えて下さい。この時、元の画像と同じ位置に画像を張り替えないと、Galleryページで、画像を表示させた場合にレイアウトが崩れるので注意が必要です。フラッシュのガイドを利用するか、元の画像が張られている、座標を参照するなどして同じ位置に張り付けて下さい。

＜Diaryについて＞
Diaryでは外部テキストを参照する形をとっています。

外部テキストファイルの記述方法やアップロードの決まり等がありますので、詳細な内容は収録素材「text_scroll01」の解説ページをご覧下さい。 画面右側に用意されたボタン42コから、日ごと、週ごと、または月ごと編集した日記を参照するという形を取られてはいかがでしょうか。
現在、Diary用のボタンはダミーとして一つしか用意されていません。Diary内で配置されている42コボタンは全て同じものですので、必要に応じて複製を作るなど、編集を行って下さい。 ボタンは ライブラリ → Diary用 → Diary用ボタン →Diary_button_0001にあります。
ボタンのテキストの編集は、同じフォルダ内の ボタン用 テキスト というシンボルを編集する事で可能です。

＜BBSについて＞
BBSページでは、各人で用意された掲示板にリンクさせて使っていただくという形をとっています。
画面左下にある"Go to BBS" ボタンにリンクをつける事によってお使いください。
現在入っているダミーのテキストには、掲示板での最近の書き込みなどを編集してのせてみてはいかがでしょうか。

＜Mailボタン＞
Mailボタンには現在仮のリンク（メールアドレス）が設定されています。
ALLの「メニューボタン」上に配置されている、ボタンシンボル "Mail" に選択の上、編集を行って下さい。

このファイル内に使っているフォントは、ArialまたはArial boldとなっています。
