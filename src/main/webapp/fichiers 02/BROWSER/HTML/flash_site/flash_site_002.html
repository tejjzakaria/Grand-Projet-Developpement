■ファイル名 flash_site_002
■ファイルサイズ 256K
■初期サイズ 656×384
■イメージ
モノトーンを基調としたシンプルな構成ながら、動きが面白いフラッシュサイトです。
■バックグラウンド色の変更
ムービープロパティを開いて背景色を変更してください。
■テキスト
＜長さ＞
とくに制限はありません。フラッシュのサイズ内でであれば自由に変更できます。
■サイトの構成と編集

＜サイトの構成＞
シーン1には、ALLというムービークリップが配置されています。この"ALL"の中には、Opening、Top、About、Gallery、Diary、BBS、の各ムービークリップ（またはグラフィックシンボル）が配置されており、 このサイトのすべてのコンテンツが含まれています。（ムービークリップ "ALL" はライブラリ内、「01.メイン」というフォルダの中にあります。）

＜編集＞
サイトのタイトルを変更するには ライブラリ内、「01.メイン」というフォルダの中にある、Site title "Sample" というシンボルを編集する事で変更可能です。 また、Topページで現在仮に入っているテキストを変更するには ライブラリ → 02.Top用 → ダミーテキスト1を編集することで可能です。

その他の各コンテンツは、ライブラリ内にそれぞれフォルダに分けて収納されていますので、編集の際にはそれぞれの該当ファイルを開き編集を行って下さい。

＜Aboutについて＞
現在入っているダミーのテキストを変更するには、ライブラリ → 03.about用 → ダミーテキスト2 を編集することで、可能です。
画像の変更は、ライブラリ → 03.about用 → Sample picture 01 の画像を張り替えることで変更可能です。

＜Galleryについて＞
Galleryでは1ページで14枚の画像を表示できる構成となっています。

画像を変更するには、ライブラリ → 04.gallery用 → gallery用イメージ フォルダー内の、pic001〜014 に、任意の画像を張り替えることで可能です。この時、元の画像と同じ位置に画像を張り替えないと、Galleryページで、レイアウトが崩れるので注意が必要です。フラッシュのガイドを利用するか、元の画像が張られている、座標を参照するなどして同じ位置に張り付けて下さい。
今回は、Contents 01、Contents 02 と、2ページ分のサンプルを用意しましたが、必要に応じて、Contents フォルダーごと複製し、各内容を編集ののち、中に含まれている gallery contents ○○ というムービークリップを"Gallery" のムービークリップ上に配置することで Contents を増やすことも可能です。
配置の仕方は、"Gallery" のムービークリップ、gallery contents 02 というレイヤーの20フレーム目から32フレーム目までと同様の構成で、33フレーム以降に作成して下さい。（この時マスクレイヤーも同じだけ伸ばす必要があります。）
ただ、あまりコンテンツを増やしていくと、75フレーム目から始まる 場面転換 01 〜04 にかかってしまうこともあるでしょう。
その場合は、フレームが重ならないように 場面転換 01 〜04を必要フレーム分後ろにずらして下さい。（後ろにずらしたことによるリンクへの影響はありません。）この時、「アクション」レイヤーと「メニューカベ」レイヤーも同じだけ後ろにずらさないと、正常に表示されなくなりますので、注意が必要です。
現在入っているダミーのテキストを変更するには、ライブラリ → 04.gallery用 → Contents 01（またはContents 02）→ ダミーテキスト3-1（または3-2）を編集することで、可能です。

＜Diaryについて＞
Diaryのトップページには 、最新の日記をのせ、過去の日記は通常のHTMLでみせるという形式になっています。
テキストの変更は、text diary用 というシンボルを編集することで変更可能です。過去の日記へのリンクは、Diary のムービークリップ上に配置している、previous (diary)というボタンシンボルにアクションを設定して下さい。（現在は仮のリンクが設定されています。）

＜BBSについて＞
BBSページでは、各人で用意された掲示板にリンクさせて使っていただくという形をとっています。
画面中央、下部にある"Go to BBS" ボタンにリンクをつける事によってお使いください。

現在入っているダミーのテキストには、掲示板での最近の書き込みなどを編集してのせてみてはいかがでしょうか。
テキストの変更は、ライブラリ → 06.bbs用 → ダミーテキスト5 を編集することで、可能です。

＜Mailボタン＞
Mailボタンには現在仮のリンク（メールアドレス）が設定されています。
変更には、各ページ上のムービークリップ（Top、About、Galery、Diary、BBS）で、「メニュー」というレイヤー上に配置されている"Mail"ボタンを 選択の上、編集を行って下さい。（５ページそれぞれのボタンに設定する必要がありますので、ご注意ください。）

このファイル内に使っているフォントは、ArialまたはArial boldとなっています。
