＜長さ＞

■サイトの構成と編集

＜サイトの構成＞
シーン1には、ALLというムービークリップが配置されています。この"ALL"の中には、Opening、Top、About、Gallery、Diary、BBS、の各ムービークリップ（またはグラフィックシンボル）が配置されており、 このサイトのすべてのコンテンツが含まれています。

＜編集＞
各ムービークリップをはじめとした、パーツはライブラリ内で、それぞれフォルダーごとに分けられておりますので、編集作業の際には、該当ファイルを開き、編集を行って下さい。 サイトのタイトル変更は、ライブラリ内、「共通パーツ」の、"Site title" というシンボルを編集する事で変更可能です。

Contents 01〜03は、ダミーとしてTop、About、BBS共通で使っています。目的に応じて、複製を作るなど、内容、配置位置などに変更を加えてお使い下さい。

＜各ボタンへのリンクの付け方＞
各ムービー上、またはグラフィックシンボル上に配置されているボタンシンボル（BBSページの"Go to BBS" や Diaryページの "More diaries") を選択し、それぞれにアクションを指定して下さい。

＜Galleryについて＞
画像はそれぞれボタンシンボル化されており、配置されたボタンにマウスを重ねる事で、画像が表示されます。 表示される画像のサイズは225×165となっていますが、もっと大きなサイズで画像を見せたい場合は、各ボタンにリンクをつける事で通常のHTMLで見せるという方法もあります。

画像を変更するには、ライブラリ → 04.Gallery用 → Gallery sample image の中の、Gallery sample image 001〜010に、任意の画像を張り替えることで可能です。 この時注意しなければならないのは、元の画像と同じ位置に張り替えないとレイアウトがずれてしまいますので、フラッシュのガイドや、元の画像が張られていた座標を参照するなどして作業を行って下さい。

＜Diaryについて＞
Diaryでは外部テキストファイルを参照する形をとっています。
外部テキストファイルの記述方法やアップロードの決まり等がありますので、詳細な内容は収録素材「text_scroll01」の解説ページをご覧下さい。

このサイトでは日記を、Monday〜Sundayまでの7つのボタン で表示させ、その他の日記はHTMLでみせる形式をとっていますが、各ボタンの使い方は自由です。週ごと、月ごとに日記をまとめるという使い方もいいかもしれません。
ボタンのテキストを変更する必要が生じた場合は、ライブラリ → 05.Diary用 → Diary ボタン用 の中の、Monday 〜 Sunday のそれぞれシンボルを編集することで変更可能です。

＜BBSについて＞
BBSページでは、各人で用意された掲示板にリンクさせて使っていただくという形をとっています。
画面中央、下部にある"Go to BBS" ボタンにリンクをつける事によってお使いください。
現在入っているダミーのテキストには、掲示板での最近の書き込みなどを編集してのせてみてはいかがでしょうか。

＜Mailボタン＞
Mailボタンには現在仮のリンク（メールアドレス）が設定されています。
変更には、各ページ上に配置されている"Mail"ボタンを 選択の上、編集を行って下さい。（５ページそれぞれのボタンに設定する必要がありますので、ご注意ください。）

このファイル内に使っているフォントは、ArialまたはArial boldとなっています。
