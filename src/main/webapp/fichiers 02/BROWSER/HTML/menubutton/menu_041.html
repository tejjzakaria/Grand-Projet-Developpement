■URL変更の説明

■その他
ボタンの数は6個用意されています。
ボタンの数を減らす場合は、
1. シーン１の「コンテンツボタン」レイヤーにあるボタンのなかで必要のないボタンを削除。
2. ライブラリ内の「コンテンツ」フォルダを開き、そのなかで必要のないボタンを削除して下さい。

ボタンの数を増やす場合は、
1.ライブラリ内の 「コンテンツ」フォルダを開き、「コンテンツ＿about」を複製して下さい。続いて、グラフィックシンボル「about」も複製して下さい。
2. さきほど複製したシンボル「aboutのコピー」を開き、テキストを変更して下さい。
3. それから、複製した「コンテンツ＿about」を開き、”アップ”、”オーバー”、”ダウン”にあるシンボルを複製した「aboutのコピー」へとシンボルの入れ替えを行って下さい。
4. シーン１の「コンテンツボタン」レイヤーにその複製して作ったボタンを配置して下さい。
