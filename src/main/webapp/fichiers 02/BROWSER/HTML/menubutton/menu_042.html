■ファイルサイズ 29K
■初期サイズ 290×110
■イメージ
透明感のある液体を表現したイメージです。
■バックグラウンド色の変更
ムービープロパティを開いて背景色を変更してください。
■テキスト
＜長さ＞
テキストの乗っている台の内に収まるところまで可能です。
＜色＞
#FFFFFFの白に設定されています。
＜変更箇所＞
ライブラリ内のシンボル「about」「gallery」「work」「diary」「bbs」「link」を開いて、それぞれテキストを変更することで完了です。
＜注意＞
FLASHファイルをカスタマイズする際、システムに入ってないフォントが使用されている場合、一部のフォントが表示されないことがあります。 表示するにはフォントを変更して下さい。

■URL変更の説明
シーン1上の「ボタン」レイヤーのオブジェクトにアクションを設定して下さい。
■その他
ボタンの数は6個用意されています。
ボタンの数を減らす場合は、
1. シーン１の「ボタン」レイヤーにあるボタンのなかで必要のないボタンを削除。
2. ライブラリ内のなかで必要のないボタンを削除して下さい。

ボタンの数を増やす場合は、
1.ライブラリ内の 「ボタン＿about」を複製して下さい。続いて、グラフィックシンボル「about」も複製して下さい。
2. さきほど複製したシンボル「aboutのコピー」を開き、テキストを変更して下さい。
3. それから、複製した「コンテンツ＿about」を開き、”アップ”、”オーバー”、”ダウン”にあるシンボルを複製した「aboutのコピー」へとシンボルの入れ替えを行って下さい。
4. シーン１の「ボタン」レイヤーにその複製して作ったボタンを配置して下さい。
